# Swin-UNet for Brain MRI Tumor Segmentation
medical image segmentation pipeline using the Swin-UNet architecture. It leverages a hierarchical Swin Transformer as the encoder and a symmetric transformer-based decoder to perform precise binary segmentation of brain tumors from FLAIR MRI scans.
